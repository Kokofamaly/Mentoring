Output: 

C# 5:
1, 1
2, 2
3, 3

C# 4:
3, 4
3, 4
3, 4



C# 4 and C# 5 differ in how loop variables are handled in closures.
In C# 5 loops a new temporary variable is created for every iteration.
in C# 4 loops closures take one final result of the variable for every iteration.


public class Program {

  public static void Main() {

    var actions = new List<Action>();

    int counter = 1;

 

    foreach (var i in Enumerable.Range(1, 3)) {

      actions.Add(() => Console.WriteLine(i + ", " + counter));

      counter++;

    }

    foreach (var action in actions) {

      action();

    }

  }

}
