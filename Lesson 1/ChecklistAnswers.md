**What does the .NET ecosystem provide?**<br>
    The .NET ecosystem provides:  
    - different runtimes such as .NET Framework, .NET Core, .NET 5 and also Mono for Xamarin;  
    - tools that are useful for apps compiling, debugging, building and publishing;  
    - libraries;  
    - languages;  <br><br>
    In short, the .NET ecosystem provides everything you need to write, test, run, and deploy 
    modern applications: desktop, web, mobile, cloud, and browser-based apps.
    
**What are .NET implementations? Which ones are used nowadays?**  
    The main .NET implementations include: .NET Core, .NET Framework, Mono.
    Nowadays .NET Core is the most used implementation.

**What is CLR?**  
    CLR(Common Language Runtime) is the runtime that helps execute code safely for the system.
    At first, it converts code to IL and then the code is being executed by using JIT.

**Why is .NET 5 a bit of a special version?**  
    .NET 5 is special because it was the first version to unify .NET Core and .NET Framework into a single and cross-platform platform.  

**Which technologies are supported by the .NET framework?**  
    The .NET Framework supports:  
    - Windows Forms and WPF for desktop applications;  
    - ASP.NET for web applications;  
    - ADO.NET and Entity Framework for data access;  
    - WCF and Web API for services and APIs;  
    - LINQ and XML for data processing;  

**Does cross-platform development is possible in .NET? What about cross-platform UI?**  
    Yes, cross-platform development is possible. Cross-platform UI is also possible, while using the right framework.

**What does the multitargeting term mean?**  
    Multitargeting means a project can be built to run on multiple .NET platforms or versions.
